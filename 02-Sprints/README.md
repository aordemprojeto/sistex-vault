# 02-Sprints

Iniciativas com **prazo definido** (ex: `MVP-Stenio-05mai`, `Bloco-13-Whitelabel`). Use o template [[../_Templates/Template-Sprint|Template-Sprint]] e mantenha um arquivo por sprint. Sprints encerradas movem para `99-Arquivo/`.
