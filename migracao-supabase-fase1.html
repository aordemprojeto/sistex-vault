Suas 5 ações no Supabase Dashboard

1

Logar no Supabase Dashboard

URL: https://supabase.com/dashboard

Clica em "Continue with GitHub" (em aba anônima do Chrome se ainda der erro de OAuth state).

2

Criar Organization (se não existir uma sua)

Quando entrar, você pode estar dentro de uma org herdada do Lovable. Clica no nome da org no topo (canto superior esquerdo) e procura "New organization".

  • Name: YNTHEGRA
  • Type: Personal (vai mudar pra Team se crescer)
  • Plan: Free (sobe quando precisar)

Se já tiver uma org sua, pula para o passo 3.

3

Criar Project

Dentro da sua organization, clica em "New project" e configura:

Namesistex
Database passwordGera senha forte (use 1Password) — SALVE com cuidado
RegionSouth America (São Paulo)
Pricing planFree

Clica em "Create new project". Aguarda ~2 minutos pelo provisionamento (banco, API, edge runtime).

4

Coletar 4 credenciais do projeto

Quando terminar de provisionar, vai em Project Settings → API. Você vai precisar copiar 4 valores:

Project Reference IDtipo xyzabc123def456 (string curta)
Project URLtipo https://xyzabc123def456.supabase.co
anon (public)JWT começando com eyJ...
service_roleJWT começando com eyJ... — SECRETO
5

Gerar Personal Access Token

URL direta: https://supabase.com/dashboard/account/tokens

  1. Clica em "Generate new token"
  2. Name: claude-code-cli
  3. Copia o token que aparece — começa com sbp_...
  4. ⚠ Você só vai ver esse token UMA VEZ. Salve no 1Password.

Como me passar as 6 credenciais

Cola aqui no chat do Claude Code, exatamente neste formato (preserva estrutura — facilita meu parse):

PROJECT_REF=xyzabc123def456 PROJECT_URL=https://xyzabc123def456.supabase.co ANON_KEY=eyJhbGc... SERVICE_ROLE_KEY=eyJhbGc... PAT=sbp_... DB_PASSWORD=a-senha-que-voce-criou-no-passo-3
⚠ Sobre os secrets que você vai compartilhar
  • SERVICE_ROLE_KEY — chave admin do banco. Nunca compartilhe em outro lugar.
  • PAT — acesso administrativo total. Após eu terminar o Dia 1, revogue em Account → Tokens.
  • DB_PASSWORD — senha do banco. Salve no 1Password antes de me mandar.
  • O chat do Claude Code é seguro pra essas trocas, mas nunca cole esses valores em e-mail, WhatsApp, ou outro chat público.
